# AI-Based Student Chatbot

This project is a chatbot built using Flask to assist students with basic queries and provide quick responses.

## Features
- Handles common student queries
- Simple and interactive interface
- Lightweight and easy to use

## Tech Stack
- Python
- Flask
- NLP (basic)

## How It Works
The chatbot processes user input and returns relevant responses based on predefined logic and simple NLP techniques.

## Future Improvements
- Improve NLP for better accuracy
- Add database integration
- Deploy as a web application
